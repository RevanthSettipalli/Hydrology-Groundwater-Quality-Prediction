# 🌍 Hydrology Groundwater Quality Prediction

AI-powered groundwater quality prediction system using Machine Learning and GIS Visualization.

## Features

✔ Data Upload (.CSV)  
✔ Automatic Data Cleaning  
✔ Decision Tree Model  
✔ Random Forest Model  
✔ Logistic Regression  
✔ SVM Classification  
✔ Model Accuracy Comparison  
✔ Confusion Matrix  
✔ ROC Curve  
✔ Risk Meter  
✔ Location-Based Prediction  
✔ Premium GIS Map Visualization  

---

## Technologies

- Python
- Google Colab
- Pandas
- NumPy
- Scikit-Learn
- Plotly
- GeoPandas
- GeoPy

## Installation

```bash
pip install -r requirements.txt
```

Open:

```bash

```
